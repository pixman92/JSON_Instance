# JSON_Instance

A Javascript Class file, for making new and separate JSON objects.

Declaring instance
```
var myJSON = new JSON_Instance()
```


Adding 2d Array to the JSON
```
myJSON.addToObj(<2dArray>)
```

To print to console
```
myJSON.print()
```

Returns a JSON string
```
myJSON.stringMe()
```

Parses the JSON string
```
myJSON.parseMe()
```
